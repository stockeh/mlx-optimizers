A library to experiment with new optimization algorithms in [MLX](https://github.com/ml-explore/mlx).

```python
import mlx_optimizers as optim

#... model, grads, etc.
optimizer = optim.DiffGrad(learning_rate=0.001)
optimizer.update(model, grads)
```
